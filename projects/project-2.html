Miata digital-twin

In Progress 2025 - Present

Building a live digital twin of a car, streaming real-world sensor data scanned from OBD-II to a Unity simulation.

View Code

Project Overview

This project creates a high-fidelity digital twin of my 1999 Mazda Miata. A Raspberry Pi 5 collects live data from the car's OBD-II port, GPS, and battery sensors, streaming it via MQTT to a ThingsBoard dashboard and a physics-based 3D model in the Unity game engine. The goal is to enable real-time monitoring, performance analysis, and predictive modeling of component wear, bridging physical hardware with a live virtual simulation.

Technologies

IoT Protocols & Platforms (MQTT & ThingsBoard) Game Engine Development (Unity) Digital Twin & Simulation Modeling
