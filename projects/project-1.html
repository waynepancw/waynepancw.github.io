spot-mini

In Progress 2023 - present

Adapting an open-source quadruped, bridging direct hardware control with custom reinforcement learning policies.

View Code

Project Overview

This project involves building a Mini Cheetah-style quadruped, adapting the core framework and kinematics from the moribots/spot_mini_mini open-source project. My focus is on the full development pipeline, from implementing direct hardware control on a Raspberry Pi to training custom reinforcement learning policies in PyBullet for robust sim-to-real transfer. The project currently involves adapting the initial systems for manual control as a foundation for future autonomous behaviors.
